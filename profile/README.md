# 武汉科技大学 ACM 俱乐部（WUST ACM Club）

> 面向算法竞赛与工程能力的长期训练与开源沉淀。  
> 我们代表武汉科技大学参加 XCPC（ICPC / CCPC）系列赛事，长期保持稳定获牌与进步。

---

## 我们是谁

武汉科技大学 ACM 俱乐部（WUST ACM Club）是以 *程序设计竞赛 (Competitive Programming)* 为核心的学生技术社团与集训团队，聚焦：

- 数据结构与算法训练
- XCPC（ICPC / CCPC）体系化备赛
- 代码规范、工程实践与团队协作
- 知识库沉淀与开源共享

我们相信：持续训练 + 复盘迭代 + 团队协作 + 开源沉淀，才是长期强队的底层逻辑。

## 我们怎么训练

- **周训 / 专题训练**：按难度与知识点拆分（图论、DP、数据结构、数论、计算几何、字符串等）
- **模拟赛 / 复盘**：赛后复盘比“刷题”更重要（题意抽象、模型选择、复杂度、实现细节）
- **题解与代码评审**：统一代码风格与模板，降低比赛实现风险
- **知识库建设**：把“会了”变成“随时能用”

## 贡献指南（Contributing）

- 推荐使用 `Pull Request` 提交内容（题解、笔记、模板、工具）
- 提交前请自测：样例 / 边界 / 随机对拍（如适用）
- 题解建议包含：**题意抽象 → 算法选择 → 正确性要点 → 复杂度 → 易错点**
